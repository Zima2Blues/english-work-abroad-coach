# Learning Science

Use these rules when creating or revising the user's English study plan.

## Core Principles

1. **Spaced practice**
   - Review expressions across days instead of cramming once.
   - Recycle yesterday's corrected phrases in today's speaking task.
   - Add weekly and 50-day review days.

2. **Retrieval practice**
   - Ask the user to produce language from memory before showing the polished answer.
   - Prefer "say it first, then correct it" over passive reading.
   - End each session with 3-5 expressions the user must reuse later.

3. **Comprehensible input**
   - Keep materials around B1-B2 at the beginning, with occasional B2-C1 stretch material.
   - The user passed CET-6, so do not overfocus on basic grammar drills.
   - Use short clips or passages that can be replayed, shadowed, and retold.

4. **Output and feedback loop**
   - Every day needs one output artifact: recording transcript, short spoken answer, or workplace writing.
   - Feedback should correct high-impact errors and provide one improved natural version.
   - Do not overwhelm the user with every minor mistake.

5. **Transfer to work-abroad contexts**
   - Practice self-introduction, current work, project stories, meetings, requirements clarification, progress updates, problem explanation, disagreement, help requests, and interviews.
   - Prefer reusable sentence frames over isolated vocabulary.

## Session Shape

30-minute weekday:

| Block | Minutes | Purpose |
| --- | ---: | --- |
| Duolingo warm-up | 5 | maintain habit |
| Listening | 8 | comprehensible input |
| Shadowing | 7 | pronunciation and rhythm |
| Speaking output | 7 | retrieval and fluency |
| Review | 3 | spaced phrase recall |

60-minute weekend:

| Block | Minutes | Purpose |
| --- | ---: | --- |
| Duolingo + review | 10 | habit and retrieval |
| Intensive listening | 15 | deeper input |
| Shadowing + retelling | 15 | fluency and memory |
| Writing | 15 | workplace output |
| Reflection | 5 | error tracking |
