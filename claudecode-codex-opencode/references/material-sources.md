# Material Sources

Use current web materials when available. If web access is unavailable, generate a short B2-level fallback passage and clearly label it as generated.

## Preferred Sources

| Source | Best Use | Link |
| --- | --- | --- |
| VOA Learning English | graded listening and slow-to-medium speech | https://learningenglish.voanews.com/ |
| British Council LearnEnglish | listening, speaking, writing, grammar practice | https://learnenglish.britishcouncil.org/ |
| Cambridge English activities | leveled B1-C1 exercises | https://www.cambridgeenglish.org/learning-english/activities-for-learners/ |
| BBC Learning English | phrases, pronunciation, workplace-style clips | https://www.bbc.co.uk/learningenglish |
| TED-Ed or short talks | stretch listening and retelling | https://ed.ted.com/ |

## Selection Rules

- Pick short materials first: 1-5 minutes of audio or 120-300 words of text.
- Prefer materials with transcript, captions, exercises, or clear replay value.
- Match the weekly theme when possible.
- For workday 30-minute sessions, avoid long videos and dense articles.
- For weekends, choose one slightly harder material and use it for listening, retelling, and writing.

## Search Patterns

Use these search patterns when browsing:

```text
VOA Learning English B2 work interview
British Council LearnEnglish workplace communication B2
Cambridge English B2 speaking activities work
BBC Learning English office English progress update
English project presentation B2 transcript
```

## Fallback Material Template

When generating fallback material:

- Level: B2 unless the user asks for easier or harder.
- Length: 120-180 words for weekdays, 220-320 words for weekends.
- Topic: match the day theme.
- Include 5 reusable expressions.
- Add one speaking prompt and one writing prompt.
